# Task Manager

This is a modern React single page application that utilizes AWS Serverless functions written in Typescript along with long term AWS DynamoDB storage for tasks.

## AWS Lambdas

### Login

`POST /api/auth/login`

### Get Users

`GET /api/users`

### Create User

`POST /api/users``

### Delete User

`DELETE /api/users/:id`

### Get tasks

`GET /api/tasks`

### Create task

`POST /api/tasks`

### Delete task

`DELETE /api/tasks/:id
